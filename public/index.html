What I've built

myFlix (React)

A MERN stack web app where users are able to get information on movies, genres and directors. After creating an account, users can edit their user details as well as create and edit a list of their favourite movies. This front-end project works with the myFlix API.

GitHub Repo Live Link

Meet Up

This application uses the Google Calendar API to fetch upcoming events. It is a serverless, progressive web application built with React using a test-driven development approach. It uses Google OAuth2 for login, Amazon Lambda functions for hosting, and Jest, Enzyme, Cucumber and Puppeteer for code testing.

GitHub Repo Live Link

Chat App

A React Native application built with Gifted Chat that allows users to connect. Users can pick a background color, read and send messages, share images and their location. The app uses React Native Async Storage to allow users to view stored messages when offline. Message data is stored in Cloud Firestore.
